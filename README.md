# AppFormulaire — Stage Xtralog

Plateforme **VueJS / Supabase / JSON** pour **créer, gérer et exploiter** des formulaires techniques (templates), avec saisie **mobile/desktop**, **offline**, et **export PDF** (V1 simple).

## Suivi de stage

Le suivi (cadrage, décisions, roadmap, modèle JSON, schéma BDD) est centralisé dans :

 - `journal.md`

## Objectifs (V1)

 - Créer/éditer des templates de formulaires (structure en JSON)
 - Remplir un formulaire (runner) et sauvegarder les réponses
 - Fonctionner offline pour la saisie + synchroniser ensuite
 - Exporter une submission en PDF simple

## Notes

Clients industriels anonymes, formulaires existants aujourd’hui sur papier (quotidiens/hebdomadaires).
